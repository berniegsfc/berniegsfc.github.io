JavaScript Example of using the SSC Web Services

JavaScript Web Service Example

This page demonstrates using JavaScript to access the SSC Web Services. This page is not intended to be an end-user application. It is intended to show web client developers how they could use the SSC web services in their own JavaScript clients. Use your browser's view-source/debugger function (F12 with Chrome and Firefox) to view the JavaScript source code.
