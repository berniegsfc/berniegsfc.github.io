C# SSC Web Services Example

The Satellite Situation Center (SSC) Web services allow a software developer to use portions of the SSC software in their own application.  The following describes an example of how to utilize the SSC Web Services from a client program written in C# and utilizing the Web Services Enhancements (WSE) for Microsoft .NET framework. This example doesn't provide any useful, end-user functionality but is intended merely as an example for developers of client applications to examine.

Prerequisites

One of the following two alternatives is required in order to compile and execute the example program:
  • Microsoft .NET Framework 2.0 (on any supported platform) and
    • Microsoft .NET Framework 2.0 SDK
    • Microsoft WSE 3.0
  • Microsoft .NET Framework 3.0 (on any supported platform [included with Vista]) and
    • Microsoft Windows Software Development Kit (SDK) for Windows Vista (on any supported platform [there are non-Vista supported platforms])

Example

The example C# program is contained in the file WsExample.cs. In order to execute the program, save the WsExample.cs file to your computer without the .txt extension.

Instructions

The following instructions describe how to compile and execute the example C# program:
  1. Open a .NET Framework SDK Command Prompt window and execute the following commands.
  2. wsdl https://sscweb.gsfc.nasa.gov/WS/ssc/2/SatelliteSituationCenterService?wsdl
  3. csc SatelliteSituationCenterService.cs WsExample.cs
  4. WsExample
