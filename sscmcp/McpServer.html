SSC MCP Server

Introduction

The Satellite Situation Center (SSC) Model Context Protocol (MCP) server connects AI tools to SSC. This gives AI agents, assistants, and chatbots access to SSC information (satellite location and Earth magnetic field model) and functions (conjunction queries).

Access

The SSC MCP server is publicly accessible on the Internet. A user does not have to install the MCP server on their own computer.

Prerequisites

This section describes the prerequisites for accessing SSC through the MCP server with an AI agent.

  1. An AI agent that supports remote MCP servers.
  2. An large language model (LLM) application programming interface (API) access key. The user is responsible for obtaining this and any associated costs. You would typically get this from a LLM provider or proxy (e.g., OpenAI, your employer, or other organization).

Configuration

This section describes the necessary configuration of your AI agent. The details will vary for different agents but will include the following information.

  1. LLM API access key (obtained above).
  2. LLM base URL (obtained above).
  3. AI model (choose from available with your LLM API access key).
  4. SSC MCP Server URL: https://sscweb.gsfc.nasa.gov/mcp

Note: NASA employees and contractors may use the support link below for additional help with items 1 and 2 above.
